Sobre o GIT

O GitHub é uma plataforma de hospedagem na nuvem que utiliza o sistema Git para gerenciar e versionar códigos-fonte de projetos de software. Ele serve como um hub colaborativo para desenvolvedores armazenarem projetos, rastrearem alterações, trabalharem em equipe e compartilharem código aberto (open source) com o mundo.
