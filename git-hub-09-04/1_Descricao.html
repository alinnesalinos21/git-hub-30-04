Descrição do Git Hub

O GitHub é a maior plataforma de hospedagem de código-fonte e arquivos do mundo, baseada em nuvem, que utiliza o sistema de controle de versão Git. Ele permite que desenvolvedores armazenem, compartilhem, gerenciem e colaborem em projetos de software (públicos ou privados) de qualquer lugar.

Recursos principais:

  • Hospedagem de repositórios remotos;
  • Ferramentas de colaboração;
  • Integração contínua e automação;
  • Rede social para desenvolvedores.

Por que usar o Git Hub?

Além de guardar código, ele permite:

  • Trabalho em equipe: Milhares de pessoas podem editar o mesmo projeto;
  • Histórico: Você pode soltar no tempo e ver como o código era meses atrás;
  • Portfólio: Mostre seus projetos para recrutadores e para a comunidade.
